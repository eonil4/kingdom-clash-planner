# Unit Images Directory (Public Assets)

Place unit image files here. These will be served as static assets.

## Image Naming Convention

Unit images should be named using lowercase with hyphens:

- `paladin.png` for PALADIN
- `huntress.png` for HUNTRESS
- `bonebreaker.png` for BONEBREAKER
- `necromancer.png` for NECROMANCER
- `archers.png` for ARCHERS
- `iron-guards.png` for IRON GUARDS
- `sorcerers-apprentices.png` for SORCERER'S APPRENTICES (apostrophe removed)
- etc.
