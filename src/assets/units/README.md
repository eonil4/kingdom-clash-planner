# Unit Images Directory

This directory should contain unit images extracted from the game screenshots.

## Image Naming Convention

Unit images should be named using lowercase with hyphens, matching the unit names:

- `paladin.png` for PALADIN
- `huntress.png` for HUNTRESS
- `bonebreaker.png` for BONEBREAKER
- `necromancer.png` for NECROMANCER
- `archers.png` for ARCHERS
- `iron-guards.png` for IRON GUARDS
- `sorcerer's-apprentices.png` for SORCERER'S APPRENTICES
- etc.

## Image Requirements

- Format: PNG (recommended) or JPG
- Size: Recommended 128x128px or larger (will be scaled down to 64x64px in UI)
- Background: Transparent PNG preferred, or match game style
- Aspect Ratio: Square (1:1)

## How to Extract Images

1. Open the game screenshots in an image editor
2. Crop each unit card individually
3. Save with the appropriate filename matching the unit name
4. Place all images in this directory
